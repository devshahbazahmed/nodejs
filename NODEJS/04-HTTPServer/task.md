# HTTP Server Task

## Description

Build a simple http server with following features.

1. GET: `/`: Send a simple hello from server message.
2. GET: `/contact-us`: Send your email and contact number to the user.
3. POST: `/tweet`: Do a fake DB operation and send the ack that it is done.
4. GET: `/tweet`: Send all the tweets from fake DB to the server.

Also, you need to log the incoming requests with timestamps in file name `log.txt`.
